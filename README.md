# React-Redux-Small-Test-Task

## Contains

* React
* React-Router
* Redux
* React-Router-Redux
* Webpack
* Express
* ES6
* Webpack-Dev-Server

## Getting Started

```sh
$ git clone https://github.com/alexeymikhailov/React-Redux-Small-Test-Task.git
$ cd React-Redux-Small-Test-Task
$ npm install
$ npm start
$ visit `http://localhost:3000/`
```
